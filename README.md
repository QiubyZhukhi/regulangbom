# regulangbom
Script Bom untuk Nomor telkomsel

# Tutorial 
1. install modul request "PIP INSTALL REQUESTS" jika sudah ada abaikan
2. ketik Nomor dengan Awalan 085/081
3. Masukkan Jumlah FLOOD/Jumlah pesan yang akan dikirim
4. SELESAI
